# Navigation Usability Update

This version makes the sidebar usable.

## What changed

- Sidebar menu now changes the active page.
- Dashboard, Generate Script, Storyboard, Prompt Library, Content Calendar, Projects, Analytics, and Settings now show different content.
- New Content button opens Generate Script.
- Recent Generations can be clicked to reopen an output.
- Prompt Library templates can fill the generator.
- Storyboard page shows the latest generated storyboard.
- Right insight panel shows the current active page.
